This plugin require for add functionalities for startup agency theme

== Description ==
This plugin add 3 custom post types
Testimonials
Portfolio
Team
Activate plugin with startup agency theme and you can add new testimonials, portfolios and team members as well

== Installation ==
From your WordPress dashboard

Visit Plugins > Add New
Search for “Startup Agency Core”
Install and Activate Startup Agency Core from your Plugins page
